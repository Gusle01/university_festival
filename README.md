# 대학교 축제 MoneyBTI (전북은행 대학생 서포터즈)

대학교 축제 현장 참여형 `MoneyBTI` 웹페이지입니다.  
모바일 퍼스트 UI로 구성되어 있으며, 질문 응답 기반 성향 결과와 유형별 혜택 안내, 인스타 공유 기능을 포함합니다.

- 배포 URL: <https://gusle01.github.io/university_festival/>

## 주요 목적

- 대학교 축제 참여/소비 성향 테스트 제공
- 결과 기반 참여 유도(공유/재참여/혜택 탐색)
- 전북은행 대학생 서포터즈 및 팀 브랜딩 노출

## 현재 구현 기능

### 1) 메인 화면

- 시안 기반 레이아웃(배지, 큰 타이틀, 메인 비주얼, 안내 카드)
- `테스트 시작하기` 하단 고정(플로팅) 버튼
- 전진난만 메인 이미지 + 서포터즈 공식 로고 배지

### 2) 질문 화면

- 총 10문항, 문항당 4지선다
- 상단 진행률(`Qn`, `n/10`, progress bar)
- 모바일에서 한 화면에 질문과 선택지 노출되도록 타이포/간격 최적화
- `이전` 버튼으로 응답 취소 및 점수 롤백

### 3) 결과 화면

- MBTI 기반 4유형 결과
  - `EJ` 에너지 드라이버
  - `EP` 소셜 무버
  - `IJ` 전략 큐레이터
  - `IP` 감성 아카이버
- 유형별 타이틀/설명/추천 팁 동적 렌더링
- 결과 액션 버튼
  - 서포터즈 공식 인스타
  - 전진난만 팀 인스타
  - 인스타 스토리용 이미지 공유
  - 다시 테스트하기
- 인스타/공유 버튼 아이콘 적용

### 4) 혜택 찾기 화면 (`쏙뱅크`)

- 결과 화면 CTA: `나의 성향과 "쏙"맞는 혜택 찾기` (쏙 강조)
- 유형별 혜택 템플릿(EJ/EP/IJ/IP) 분기
- `‹ 결과로 돌아가기` 버튼으로 복귀
- 쏙뱅크 앱 로고 노출

### 5) 공유 이미지 생성

- 캔버스 기반 결과 이미지 생성 후 Web Share API 호출
- 미지원 환경 폴백: 이미지 다운로드 + 인스타 이동
- 텍스트 정렬/자동 축소 로직으로 잘림 이슈 보정
- 공유 이미지 문구에 서포터즈/팀 계정 포함

## 브랜딩/에셋

- 서포터즈 공식 로고, 전진난만 이미지, 쏙뱅크 로고 반영
- 결과 유형별 SVG 비주얼(4종) 분리 관리

## 기술 스택

- HTML / CSS / Vanilla JavaScript
- 정적 배포: GitHub Pages (`main`)

## 파일 구조 (핵심)

- `index.html`: 화면 구조
- `styles.css`: 전체 스타일
- `app.js`: 질문/점수/결과/공유/혜택 화면 로직
- `assets/`: 로고, 버튼 아이콘, 결과 비주얼
